# l2k
LineageII High-Five (H5) server in Kotlin language

# Installation
- Download the project
- Open the folder L2K
- Wait for synchronize
- Go to Gradle
- Tasks
- Build and run build
- Then go to project source
- Build folder
- distributions folder
- extract L2K-1.0-SNAPSHOT.zip
- open the folder and go to bin folder
- run L2K.bat file
